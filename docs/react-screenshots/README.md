# React UI — Screenshots

Application screens (frontend). Files are ordered by user flow.

## Public (no sign-in required)

### 01-login.png
Sign-in screen — Firebase email/password. Existing users are redirected to the home page.

### 02-register.png
Account creation — Firebase user + Salesforce Contact + Firestore profile mapping.

### 03-forgot-password.png
Password reset — sends a reset link via Firebase.

## Authenticated (real account data)

### 04-home.png
**Home page** — current month summary, KPI cards, search/filter, Current and Upcoming lists, FAB to add an expense.

### 05-analysis.png
**Analysis page** — total spending, monthly trends chart, category donut chart, transaction list with edit/delete actions.
